Thank you for downloading and using Rquotes

 

Quick Start Guide: Updated for version 3.1.0


Component:

Go to component menu and choose Rquotes>Categories .
Select"New". Enter a category Name in the title box and Description (description is optional) and Click Save.
This brings you back to the component menu.

Go to List Quotes > New (this brings up the new entry form)
Fill in form as required and save your work.

 

Daily number # field ----(optional) Must be a number and is used to display info in the front end in order. Also must be used to display sequential, daily,weekly etc info in the module.

Display Date field---(optional) Must be used to display info on a particular day for both the component and module

Category field ---- If you have created categories choose one.

Status field >Published or Unpublished

Creation Date> (Optional)

User>(Optiomal)

Quotes,Riddle,etc field---(optional)Enter any information you desire. It can be text or links to images or just about anything .It will display frontend and module if desired.

Author,Riddle,etc field---(optional)Enter any information you desire. It can be text or links to images or just about anything .It will display frontend and module if desired.

Notes field--- (optional)Enter any text information you desire. .It will display frontend and module if desired.

Click on Save at the top of page and the information is saved to data base.

 

To use the information in front end create a menu item as follows::

Go to Menus>menu name > new > Rquote!> List quotes by categories>

Fill in title etc.

OR

Go to Menus>menu name > new > Rquotes Component > Show one Rquote> Click on Default Layout.
Fill in the ID # of the quote you wish to display under Parameters-Basic .This will display one quote only by the ID#
***NOTE You can not set any component options here.***

Module:
If you downloaded the module (mod_rquotes) go to extensions and choose '' Module Manager","Rquotes".
Publish the module
Choose a module position on your site.
Select to show title or not.
Choose to show quotes from a text file or the DataBase.
Choose the rest of the options based on either text file od DB.

 

Text file: To use Daily display
Select a text file to use.
Select Daily to display a new quote every 24 hours (**see note**)
Choose every page load to display a random quote.

**Note** To use Daily quote you need to have a text file with 31 lines. If you have less repeat line to get 31. As of now it works on the day of the month. More than 31 can't be displayed

DB Options:
Select a Category if you entered more than one in the component.


Select Random to display a random quote.


Select Multiple Random to display more than one random quote. Use the next field to enter how many to display. Also select how many.


Select Sequential to display quotes in sequential order based on there Daily_#. You must use this field for sequential quotes to work. Also user must have cookies enable or a random quote will display.


 Select Daily, Weekly, Monthly or Yearly  to display a  quote for that time period.(**See note**)
Select How to display quotes (default,bold,italic)

**Note **To display a Daily, Weekly, Monthly or Yearly   quote you need to set up a category for that purpose and use the (Daily # ) field in the edit or new quote form to input a number. The quotes will be displayed one every selected time period according to these numbers until all quotes have been displayed. Then will start over again. If you have TWO Daily # the same within a catergory , both will be displayed for that particular period. Please make shore that the display #'s are consecutive and all quotes are published. you can only use one of these time periods.

Example if you choose to dislpay a catergory of quotes for daily you can't use another as weekly. They must be the same time period.

 Select By date to display a quote on a particular day of the year.

You can display a link after the quote if desired to take your visitor to a new page. Fill in the link option if desired. It can refer to another web page or an external link.
